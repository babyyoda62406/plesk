Recursos¶

Los recursos (presentes en Planes de servicio > seleccionar un plan > pestaña Recursos) definen los recursos del sistema que se proporcionan con la suscripción.

Directiva de sobreuso

Define qué sucede si el uso de espacio en disco y de tráfico de la suscripción alcanza los límites establecidos:

  • Sobreuso no permitido suspenderá la suscripción.

  • Se permite el sobreuso de espacio en disco y tráfico permitirá a la suscripción operar correctamente aunque se exceda el uso del espacio en disco o del tráfico.

    Puede seleccionar la opción Notificarme por email en caso de exceso de uso de forma que los usuarios recibirán un aviso en el momento en que el espacio de disco o el tráfico utilizados excedan el límite especificado (opciones Notificar cuando el uso de espacio en disco alcance y Notificar cuando el uso de tráfico alcance respectivamente). Las notificaciones se enviarán a los usuarios y/o direcciones de email especificadas en Herramientas y configuración > Notificaciones: La suscripción ha excedido los límites de uso de recursos.

  • Sobreuso permitido permitirá a la suscripción operar correctamente cuando se exceda el uso de cualquier tipo de recurso.

    Puede seleccionar Notificarme por email en caso de exceso de uso para comunicar a los usuarios que han excedido el uso de un recurso determinado. Las notificaciones se enviarán a los usuarios y/o direcciones de email especificadas en Herramientas y configuración > Notificaciones: La suscripción ha excedido los límites de uso de recursos.

Nota

la directiva de sobreuso no se aplica a los límites definidos en términos de tamaño de los buzones de correo. Por este motivo, incluso en el caso de que habilite el sobreuso, compruebe que ha asignado suficiente espacio en disco a los buzones de correo.

Espacio en disco

La cantidad total de espacio en disco asignada a la suscripción. Esta incluye el espacio en disco ocupado por todos los archivos relacionados con la suscripción: contenido de los sitios web, bases de datos, aplicaciones, buzones de correo, archivos de registro y archivos de backup.

Notificarme cuando el uso de espacio en disco alcance

Solo disponible si se permite el exceso de uso de espacio de disco y de tráfico. Esta opción determina el límite a partir del cual se notifica el exceso de uso de espacio en disco. Cuando se alcanza dicho límite, Plesk envía notificaciones a las direcciones de email de los usuarios especificadas en Herramientas y configuración > Notificaciones: La suscripción ha excedido los límites de uso de recursos.

Nota

el límite para las notificaciones puede establecerse en MB/GB/TB o en porcentaje. Cuando el límite sea persistente, puede especificarlo en MB/GB/TB (en tal caso, el límite permanecerá inalterado cuando añada un complemento que aumente el espacio en disco para la suscripción). Cuando el límite para las notificaciones sea flotante, puede especificarlo en porcentajes (en tal caso, si posteriormente se añaden complementos, el límite se recalculará de forma automática). Por ejemplo, si su espacio de disco es = 10Gb y establece un límite para notificaciones = 9Gb y además incluye 5 Gb de espacio de disco adicional, la notificación se enviará cuando se haya excedido el límite de 9 Gb. Si, en el mismo caso, establece un límite = 90 %, la notificación se enviará cuando se haya excedido el límite de 13,5Gb.

Tráfico

La cantidad de datos que puede transferirse desde los sitios web de la suscripción y los recursos compartidos FTP/Samba durante un mes.

Notificarme cuando el uso de tráfico alcance

Solo disponible si se permite el exceso de uso de espacio de disco y de tráfico. Esta opción determina el límite a partir del cual se notifica el exceso de uso de tráfico. Cuando se alcanza dicho límite, Plesk envía notificaciones a las direcciones de email de los usuarios especificadas en Herramientas y configuración > Notificaciones: La suscripción ha excedido los límites de uso de recursos.

Sitios publicados con Presence Builder

El número de sitios web que pueden publicarse con Presence Builder.

Dominios

El número total de nombres de dominio que el suscriptor puede alojar en la suscripción. Esto incluye sitios web, configuraciones de redireccionamiento web que apuntan a sitios web alojados en otros servidores y nombres de dominio donde aún no se haya creado un sitio web o configurado el redireccionamiento web (dominios sin hosting).

Subdominios

El número total de subdominios que el suscriptor puede alojar en la suscripción.

Alias de dominio

El número total de nombres de dominio adicionales y alternativos que el suscriptor puede alojar en la suscripción.

Buzones de correo

El número total de buzones de correo que el suscriptor puede alojar en la suscripción.

Tamaño del buzón de correo

La cantidad de espacio en disco asignada a cada buzón de correo en una suscripción para el almacenamiento de emails y archivos adjuntos de respuestas automáticas.

Cuota total de buzones de correo (Hosting Windows)

La cantidad total de espacio en disco en megabytes disponible para todos los buzones de correo de la suscripción.

Listas de correo

El número total de listas de correo que el suscriptor puede alojar en la suscripción.

Cuentas FTP adicionales

El número máximo de cuentas FTP usadas para acceder a los archivos y carpetas creadas en una suscripción. Este número no incluye una cuenta que siempre se genera durante la creación de una suscripción.

Bases de datos (Hosting Unix)

El número total de bases de datos que pueden crearse en los servidores de base de datos de Plesk y ser usados por los sitios web de la suscripción.

Bases de datos MySQL databases y bases de datos MS SQL (Hosting Windows)

El número máximo de bases de datos MySQL y Microsoft SQL Server respectivamente que pueden crearse en los servidores de base de datos de Plesk y ser utilizados por los sitios web de la suscripción.

Cuota total de bases de datos MySQL y Cuota total de bases de datos MS SQL (Hosting Windows)

La cantidad máxima de espacio en disco que pueden ocupar las bases de datos MySQL o Microsoft SQL Server en una suscripción.

Tamaño del archivo de una base de datos MS SQL (Hosting Windows)

Se trata del tamaño máximo del archivo de una base de datos Microsoft SQL Server. Tenga en cuenta que si una base de datos se compone de más de un archivo, puede que se utilice más espacio en disco del especificado aquí.

Tamaño del archivo de registro de una base de datos MS SQL (Hosting Windows)

Se trata del tamaño máximo del archivo de registro de una base de datos Microsoft SQL Server. Tenga en cuenta que si una base de datos se compone de más de un archivo de registro, puede que se utilice más espacio en disco del especificado aquí.

Periodo de validez/fecha de expiración

El periodo de validez de una suscripción.

En las propiedades de un plan de servicio, esto se denomina Periodo de validez y se utiliza para las suscripciones nuevas.La fecha de expiración de una suscripción depende de este periodo.

En las propiedades de la suscripción, esto se denomina Fecha de expiración. En esta fecha, la suscripción será suspendida, lo que significa que todos los sitios de la suscripción serán suspendidos y los usuarios de Internet ya no podrán acceder a su web ni a sus servicios de correo y FTP. Asimismo, los suscriptores y sus usuarios ya no podrán acceder al panel del cliente.

Las suscripciones no se renuevan automáticamente, por lo que para reactivar los servicios de una suscripción deberá activar la suscripción de forma manual.

Aplicaciones en Java

El número total de aplicaciones Java que pueden alojarse en los sitios web de la suscripción.

Usuarios web

El número total de páginas web personales que el suscriptor puede alojar para otros usuarios en sus sitios web. Este servicio se usa principalmente en el caso de instituciones académicas que alojan páginas personales no comerciales para sus empleados y sus alumnos. Estas páginas suelen tener direcciones como http://example.com/~webuser.

Conexiones ODBC (Hosting Windows)

El número total de conexiones ODBC que el suscriptor puede usar en la suscripción.
