Cuenta FTP adicional creada¶

Nota

Este evento sólo es relevante en el caso de cuentas FTP adicionales, es decir, cuentas generadas tras la haberse creado la suscripción como cuentas complementarias a la cuenta FTP principal. Si desea realizar un seguimiento de la creación de la cuenta FTP principal, es decir, la cuenta que se crea de forma simultánea cuando se crea una suscripción, debería utilizar el evento Configuración de hosting creada.

Nombre y descripción del parámetro Nombre de variable de entorno Notas
Nombre de dominio NEW_DOMAIN_NAME Obligatorio
Directorio principal NEW_HOME_DIRECTORY Obligatorio
Nombre de usuario del sistema NEW_SYSTEM_USER  
Contraseña de usuario de sistema NEW_SYSTEM_USER_PASSWORD
